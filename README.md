
# Lisp-Desu

Toy lisp implementation.

### Quickstart

```console
$ cargo build --release --bin=lisp-desu
$ ./target/release/lisp-desu [-o <OUTPUT PATH>] <INPUT PATH>
```

Check out examples in [lisp-desu](lisp-desu/examples).
